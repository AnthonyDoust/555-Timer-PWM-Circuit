# 555-Timer-PWM-Circuit
This circuit creates a ~1kHz PWM signal that ranges from 10% to 99% duty cycle. I have made this to explore creating PWM signals without the use of a microcontroller. For uses in DC-DC converters, Motor speed control, LED brightness control, ect. Attached is the circuit schematic and an example picture using the circuit to drive a small DC motor.

Parts List:

1x SA555P

3x 0.1uF Cermamic Capacitors

2x 1N4148 Signal Diodes

1x 1k Resistor

1x 10k Potentiometer
